v1.0.0
使用指南
✕
1
设置游戏窗口
建议把程序放在英文路径；如果中文路径下 OCR 或启动异常，再移动整个文件夹。游戏视频设置建议使用无边框窗口或窗口化无边框，独占全屏可能让截图或翻译框显示不稳定。
2
创建 DeepSeek API Key
打开 DeepSeek 开放平台的 API Keys 页面，点击创建 API key。创建后只会显示一次，请立即复制。
3
填入程序并获取模型
服务选择 DeepSeek，API URL 使用 https://api.deepseek.com，把刚复制的 API Key 粘贴进去。点击"获取模型"后选择 deepseek-v4-flash。

注意：DeepSeek API 按量计费，需要在平台充值余额后才能稳定使用；聊天翻译用量通常很小，实际费用非常便宜。
4
设置翻译框
程序已经带有一组 游戏聊天区域和翻译框默认位置。先直接试用；如果识别漏字、位置不准或翻译明显不对，再点击"选择翻译区域"重新框选 游戏左侧聊天框。

"调整翻译框"用于移动或缩放译文显示位置；"常态显示翻译框"会让翻译框一直显示；关闭"鼠标穿透"后可以拖动、缩放和滚动翻译框。
5
实机效果
正常游玩时翻译框会保持低存在感，不遮挡主要视野。打开聊天框时，历史消息也会按聊天顺序显示，方便回看。
6
中文回话翻译复制
开启"显示回话输入条"后，翻译框底部会出现输入栏。输入中文，选择自动、英语、日语或韩语，按 Enter 后程序会把译文复制到剪贴板。

程序不会自动替你发送游戏聊天。翻译完成后，请在 游戏聊天框里粘贴，再由你自己按 Enter 发送。
