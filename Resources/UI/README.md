# UI Asset Candidates

These are candidate assets for later UI polish. They are not wired into the app yet.

- `game-translator-lens-icon.png`: 1024 px O/T app icon with game-inspired orange, white, charcoal, and cyan accents.
- `game-translator-lens-icon.ico`: Windows app icon generated from the PNG in common icon sizes.
- `reply-translate.svg`: reply translation action icon concept.
- `ocr-scan.svg`: OCR scan mode icon concept.
- `overlay-window.svg`: overlay window icon concept.

Design direction: dark dark game-style utility surface, cyan OCR accents, soft green translation accents, no game logos or copyrighted marks.
