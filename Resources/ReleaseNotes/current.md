# Game Translator Lens v1.1.0

## 更新要点

- 新增手动模式：如果担心自动 OCR 影响帧率，可以切到手动模式，只在需要时识别一次。
- 翻译框新增相机按钮：点击即可立即识别当前框选区域；自动模式下也能使用。
- 手动模式支持 `Ctrl+Shift+T` 快捷键，按一下就识别一次。
- 自动识别更省资源：空闲时降低 OCR 频率，看到聊天后再短时间加快识别。
- 回话翻译更稳：剪贴板被微信等软件占用时，会把译文留在输入框里，方便手动复制。
- 修复多屏和屏幕边缘框选导致的区域错误、连续报错问题。
- 优化浅色主题和 overlay 下拉菜单显示，文字更清楚。
- 修复 overlay 回话条语言选择按钮背景不随透明度变化的问题。
- 新增更新后说明弹窗和一键卸载器。
